Version history

7 Febuary 2022 Version 3

  • https://acas2-alpha.herokuapp.com
    Username: acas
    Password: acas999

  • Made changes based on findings from user testing sessions with Claimants and Professionals from 24 January 2022 to 4 Febuary 2022.

    1. Based on user testing, we have removed a link to Companies House information on GOV.UK, if they have ‘Plc’, ‘ltd’ or ‘Limited’ at the end of their name, to lookup organisation's legal name and address under hint text.
    2. We shared our resaerch on GDS design system 'Task list pages' discussion on GitHub . Based on recommendation we are experimenting with adding something like "You have completed X of Y required sections [and U of V optional sections]"
    3. Based on negative emotional reaction from users, the questions "Can we speak to them(organistaion or person the dsipute is against) before we speak to you?" has been reomved. One user said this would create a panic and most likely, stop her submiting the form. One user said "this is creepy" and many others had long pause, didn't understand why this question was been asked.
    4. Based on feedback from user testing and interviews with ECSOs we got evidence that users are likely to enter more information than they need to for some Textare input feilds. So, we have added the "Character count" pattern from the GDS design system, where there was good reason for limiting the number of characters users can enter for some questions. At first we tried textarea without the limiting the characters, but feedback from user testing showed that some users said they could go town and start writing "war and peace" here.
    5. Made more content changes to questions based user testing sessions and recommendations from ECSOs and conciliators.
    6. In process of making more changes based on findings and recommendations from the new DAC accessibility test report in February.

24 January 2022 Version 2

  • https://acas2-alpha.herokuapp.com
    Username: acas
    Password: acas999

  • Made changes based on findings from user testing sessions with Claimants and Professionals from 10 January 2022 to 21 Febuary 2022.

    1. To help user with Accessibility needs we have added we have added "Need help" section in the footer. This has a link to helpline. They’ll fill in the form for the user over the phone. Also, there is link to download the paper version of the form.
    2. We are trying an alternate version for question about "What is the reason for your dispute?" which has checkboxes instead od radio to make it easier for users to select multiple reasons for your dispute.
    3. We are trying a variation of the start page, which has has the start button higher up, based on user feedback and shows the step by step journey for users that want to try early conciliation.
    4. Important information about time limits has been added.
    5. Made more changes based on findings and recommendations from the new DAC accessibility report in December.
    6. Made more content changes to questions based user testing sessions and recommendations from ECSOs and conciliators.
    7. Created another version of the user journeys without the tasklist pattern. User were asked to test and compare, after testing both prototypes. One with the task list pattern and one without the task list pattern. User cleared said they preferred the version with the tasklist pattern.

29 December 2021 Version 1.2

  • https://acas-alpha.herokuapp.com/
    Username: acas
    Password: acas777

  • Made changes based on findings from user testing sessions with Claimants and Professionals from 9 December 2021 to 23 December 2021.

    1. Based feedback from professional reprentative we added warning "If early conciliation is not successful you can still go to the employment tribunal." to "Why try early conciliation?" under "Would you like to try the free early conciliation service?" question.
    2. Changed hint text for Legal Name to "Enter legal name as it appears on your passport, birth certificate or driving licence" from "Enter legal name as it appears on your passport" as one of the professional rep pointed out that some users may not have a possport.
    3. Removed the word "jeopardising" as users found it stressful and paused with an emotional reaction when they read this word.
    4. Added "recent" before payslip and removed "letter of employment and contract" in the hint text for legal name and address, as professionals said that some times the name of company name may have changed over the years.
    5. Added "P40 and P60" to the hint text as most of the time users can get the legal name and address from their P40 and P60.
    6. Added a link for looking for the name of a school, on Information about schools on GOV.UK, which was a professional users said they use to verify the address is correct.
    7. Added a link on GOV.UK to a tool to calculate holiday entitlement to the help text under question "How much money do you think you are owed?"
